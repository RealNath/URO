# Program Pertarungan Robot
Program ini ditulis dalam bahasa Python.

## CARA MAIN
1. Jalankan programnya.
    > `python Jawaban_2.py`
2. Pilih robot pertama dan robot kedua.
3. Tonton pertarungan sengit nya.
